# Information

## Build

`dx build --package web`
`dx build --package desktop`
`dx build --package mobile`

## Typecheck

`cargo check --workspace`

## Serve

`dx serve --package web`
`dx serve --package desktop`
`dx serve --package mobile`

## Format

`dx fmt`
